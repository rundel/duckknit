# Changelog

## duckknit 0.0.9.9000

- Initial release.
- [duckdb](https://r.duckdb.org/) knitr engine with persistent CLI
  sessions across chunks.
- Chunk options: `db`, `session`, `mode`, `timeout`, `ansi`.
- [`duckdb_sitrep()`](https://rundel.github.io/duckknit/reference/duckdb_sitrep.md)
  for diagnostics.
